# MVC CoolScript Stable

This project is a web-based application built using the MVC (Model-View-Controller) architecture.

## Web Access

You can access the application at: [https://mvc.coolscript.hu](https://mvc.coolscript.hu)

## Features

- Modern MVC architecture
- Stable and reliable
- Easy to use web interface

## Getting Started

Clone the repository and follow the setup instructions to run locally.
