Installation Guide

  1. Requirements:
    • PHP 8.2 or newer
    • MySQL (optional, only if using database features)
    • Apache with mod_rewrite (for clean URLs)
  2. Setup:
    • Clone or extract the framework files into your project directory.
    • Set your web server's document root to the public/ folder.
    • Ensure public/.htaccess is present for URL rewriting.
  3. Configuration:
    • Edit app/config/settings.php for your application settings.
    • Define your routes in app/config/routes.php.
    • If using database, edit app/config/database.php with your credentials.
  4. Creating Controllers:
    • Add controllers to app/Controllers/ extending System\Core\Render.
    • Define routes for your controllers in routes.php.
  5. Creating Views:
    • Add views to app/Views/.
    • Optionally create subdirectory-specific templates (e.g., app/Views/user/template.php).
    • Create a global template at app/Views/template.php or app/Views/layout.php.
  6. Template System:
    • Framework automatically detects templates in this order:
    • 1. app/Views/{subdirectory}/template.php
    • 2. app/Views/{subdirectory}/layout.php
    • 3. app/Views/template.php
    • 4. app/Views/layout.php
    • View content is available as $viewContent in the template.

For more details, see Feature List and Main Documentation.
