Feature List

  • Route-Based System: All routes defined in routes.php
  • Automatic Template Detection: Checks for subdirectory templates first, then global template
  • Bootstrap 5 Integration: Pre-configured for rapid development
  • PSR-4 Autoloading: For App and System namespaces
  • View Rendering: Automatic template/layout wrapping with $viewContent variable
  • 404 Fallback: Automatic fallback to 404.php for missing views
  • Configuration System: Centralized settings in settings.php
  • Database Support: Optional database configuration
  • Clean URLs: Via .htaccess URL rewriting
  • Secure Access: Only public folder is web-accessible

Framework Structure

  • Controllers: Handle business logic, extend System\Core\Render
  • Views: Template files with optional subdirectory-specific layouts
  • Routes: Centralized route definitions
  • Configuration: Separate files for settings, routes, and database

See Installation Guide and Main Documentation for details.
